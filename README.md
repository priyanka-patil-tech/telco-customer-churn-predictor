## How to Run

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Train models (generates all artifacts in `/models`)
```bash
python train_models.py
```
Training time: ~3–5 minutes (GridSearchCV for all models).

### 3. Launch the Streamlit app
```bash
streamlit run streamlit_app.py
```
App will open at `http://localhost:8501`

## Key Findings
1. **Tenure** is the strongest churn predictor — short-tenure customers churn at 3× the rate
2. **Month-to-month contracts** have ~43% churn rate vs ~3% for two-year contracts
3. **Fiber optic** customers churn significantly more than DSL customers
4. **Electronic check** payment users churn at ~45% — highest of all payment methods
5. All tree-based models achieve AUC-ROC of 0.83–0.84 with class-balanced training
